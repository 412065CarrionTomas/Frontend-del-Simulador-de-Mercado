Sobre este Frontend

Este frontend hace alusión al influencer TJR, quien fue mi mentor en los conocimientos que poseo sobre trading.
El objetivo principal de este proyecto es demostrar conocimiento técnico, no priorizar la estética visual.

El entorno es simulado y destinado exclusivamente a pruebas, por lo que se utiliza su imagen de manera libre y sin fines comerciales.

Asimismo, la propuesta visual busca reflejar la personalidad de TJR. Por este motivo, algunos elementos pueden parecer poco convencionales o incluso poco profesionales desde un enfoque corporativo tradicional, lo cual es una decisión intencional.
